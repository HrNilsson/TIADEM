Interface: TimeSyncNotify

interface TimeSyncNotify

time sync module (TimeSyncM) provides notification of arriving and transmitted time-sync msgs through TimeSyncNotify interface:

Events
event void msg_received() fired when time-sync msg is received and accepted
event void msg_sent() fired when time-sync msg is sent by TimeSyncM or the sending did not succeed
