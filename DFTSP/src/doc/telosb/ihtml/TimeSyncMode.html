Interface: TimeSyncMode

interface TimeSyncMode

the time sync module can work in two modes: - TS_TIMER_MODE (default): TS msgs sent period. from the timer - TS_USER_MODE: TS msgs sent only when explic. asked by user via TimeSyncMode.send() command, TimeSync.Timer is stopped in this mode

Commands
command uint8_t getMode() Gets the current mode of the TimeSync module.
command error_t send() command to send out time synchronization message.
command error_t setMode(uint8_t mode) Sets the current mode of the TimeSync module.
